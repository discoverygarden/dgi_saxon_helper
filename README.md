# DGI Saxon Helper

## Introduction

Helper module for running XSLT 2.0 transformations with Saxon-B.

## Requirements

Saxon-B must be installed.

## Installation

Install as usual, see
[this](https://www.drupal.org/docs/extending-drupal/installing-modules) for
further information.
